業務内容・料金

相続税・節税対策を中心に、企業の税務顧問から国際税務まで幅広く対応いたします。

相続税申告・相続手続

遺産分割協議書の作成、預貯金の名義変更、不動産の名義変更サポートまで、ワンストップで対応します。

対応業務

  • ✓ 相続税申告書の作成・提出
  • ✓ 遺産分割協議書の作成
  • ✓ 預貯金の名義変更サポート
  • ✓ 不動産の名義変更サポート
  • ✓ 戸籍・残高証明等の書類収集
  • ✓ 税務署との折衝・税務調査対応

相続税節税対策

お客様の財産状況を分析し、最適な節税プランをご提案。税務署寄りでない積極的な姿勢で、とことんまで節税を追求します。

🏠 土地の有効活用

借入金を活用した土地活用、土地の分割による評価減など、不動産に関する節税対策をご提案します。

🎁 生前贈与の活用

暦年贈与・教育資金贈与・住宅取得資金贈与など、ご家族の状況に合わせた贈与プランを設計します。

🛡 生命保険の活用

非課税枠を活用した生命保険の見直しにより、相続税の負担を軽減するプランをご提案します。

相続税還付請求

過去に申告した相続税、払い過ぎていませんか？土地評価の見直しにより還付を受けられるケースがあります。

完全成功報酬型 — 還付がなければ費用はかかりません

相続税の申告から5年以内であれば、更正の請求により税金が戻ってくる可能性があります。
特に土地の評価は、現地調査による個別事情の反映で大幅に減額できるケースが多くあります。

報酬：還付額の20%（還付が実現しなかった場合は無料）

賃貸不動産オーナー向けサービス

記帳代行・確定申告から相続対策・空室対策まで、月額定額でトータルサポートします。

サービス内容月額料金
記帳代行・確定申告・相続対策・空室対策（2棟まで）月額 30,000円〜
追加物件（3棟目以降）別途お見積り
